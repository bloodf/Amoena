# Features

Lunaria provides a comprehensive set of features for AI-assisted software development.

## Core

- [Sessions](./sessions.md) — Multi-turn AI conversation management
- [Providers](./providers.md) — Multi-provider AI integration (Claude, Codex, Gemini, OpenCode)
- [Routing](./routing.md) — Intelligent model selection and request routing
- [Tools](./tools.md) — Permission-brokered tool execution with approval workflows

## Intelligence

- [Agents](./agents.md) — Multi-agent orchestration with subagent spawning and team coordination
- [Autopilot](./autopilot.md) — Autonomous multi-turn task execution with phase tracking
- [Memory](./memory.md) — L0/L1/L2 observation hierarchy with embedding-based retrieval

## Infrastructure

- [Workspaces](./workspaces.md) — Git-integrated workspaces with branch isolation and merge reviews
- [Terminal](./terminal.md) — Built-in PTY-based terminal multiplexing
- [Remote Access](./remote-access.md) — Mobile device pairing via QR/PIN for remote session control
