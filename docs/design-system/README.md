# Design System

The Lunaria design system is implemented in `packages/ui/` using shadcn/ui + Radix UI primitives with Tailwind CSS v4 and an OKLCH color palette.

## Documentation

- **Component catalog:** Run `bun run --cwd packages/ui storybook` to browse components
- **Design tokens:** See `packages/tokens/` for the token definitions
- **Theming:** See `docs/architecture/theming-design-system.md` for the theming architecture
- **Accessibility:** See `docs/architecture/accessibility.md` for WCAG 2.1 AA compliance

This directory does not contain component documentation — use Storybook as the living reference.
