# Amoena v1.0.0 — Terminal Agent Control Plane

**Release Date:** 2026-03-24

Amoena v1.0.0 is the first stable release of the terminal-native AI agent control plane. It delivers a complete platform for multi-agent orchestration, extensibility, and developer productivity across desktop, TUI, and mobile interfaces.

## Highlights

- **Mission Control** — Multi-agent orchestration with goal templates, DAG execution engine, agent leaderboard, run comparison, and shareable reports with automatic secret scrubbing.
- **CQRS Event-Sourced Engine** — Command/query separation with full event sourcing for deterministic replay, audit trails, and state reconstruction.
- **Terminal UI (TUI)** — Standalone and server dual-mode terminal interface for agent interaction, session management, and real-time streaming.
- **Extension Platform** — Isolated-vm sandboxed runtime supporting the single `.luna` file format with manifest-driven contributions (commands, menus, panels, settings, hooks, tools, providers).
- **5-Language i18n** — Full internationalization support for English, Portuguese, Spanish, French, and German across desktop and mobile apps.

## What's New

### Core Platform

- `@lunaria/contracts` shared type system for cross-package type safety
- CQRS event-sourced orchestration engine with command handlers, event stores, and projections
- Settings persistence with workspace-scoped and global-scoped configuration
- Usage telemetry and cost tracking across all AI providers
- SQLite persistence with 20+ repository modules and full migration system

### Agent System

- Multi-agent orchestration with spawn, team formation, and mailbox communication
- Real Gemini adapter for Google AI model integration
- OAuth browser flow for provider authentication
- Intelligent routing service for provider/model selection based on task type
- Autopilot mode for autonomous multi-turn task execution

### Terminal UI

- Standalone mode for direct local usage
- Server mode for remote access and sharing
- PTY-based terminal multiplexing with resize support

### Extension Runtime

- Isolated-vm sandbox for secure extension execution
- Single `.luna` binary format with manifest-driven contributions
- Plugin registry for install, execute, and lifecycle management
- Rich lifecycle hooks across 23 event types

### Remote Access

- Remote access relay with UDP LAN discovery
- Device pairing via QR code and PIN
- Secure relay protocol for cross-network access

### Quality & Documentation

- 80%+ test coverage across all packages (267 Rust tests, comprehensive TypeScript test suites)
- Full API and architecture documentation
- WCAG AA accessibility compliance
- Storybook visual component development

## Breaking Changes

None — this is the first stable release. All APIs are considered stable as of v1.0.0.
