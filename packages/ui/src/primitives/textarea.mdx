## Accessibility

- Renders a native `<textarea>` element with full keyboard and screen reader support
- Focus ring visible on keyboard navigation via `focus-visible` styles
- Disabled and read-only states are natively supported
- Pair with `<Label htmlFor>` to associate a visible label for screen readers
