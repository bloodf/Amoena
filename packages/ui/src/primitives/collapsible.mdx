# Collapsible

An interactive section that can be expanded or collapsed to reveal hidden content. Built on Radix UI Collapsible primitive.

## Import

## Accessibility

- Trigger toggles content visibility via `aria-expanded`
- Supports `Enter` and `Space` keys on the trigger to toggle
- Content is removed from the accessibility tree when collapsed
- Use `disabled` prop to prevent interaction
