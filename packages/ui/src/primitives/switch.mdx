# Switch

A toggle control for switching between on and off states. Built on Radix UI's Switch primitive with a sliding thumb indicator.

## Import

## Accessibility

- Built on `@radix-ui/react-switch` which provides `role="switch"` and manages `aria-checked` state
- Toggled via `Space` or `Enter` keys
- Focus ring visible on keyboard navigation
- Disabled state reduces opacity and prevents interaction
- Pair with `<Label>` for a visible accessible label
