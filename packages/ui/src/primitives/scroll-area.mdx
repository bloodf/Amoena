# ScrollArea

A custom-styled scrollable container that replaces the browser's native scrollbar. Built on Radix UI's ScrollArea primitive with vertical and horizontal scrollbar support.

## Import

```tsx
import { ScrollArea, ScrollBar } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-scroll-area` which preserves native scrolling behavior and keyboard navigation
- Content remains accessible via keyboard (`Tab`, arrow keys, `Page Up`/`Page Down`)
- Custom scrollbars are decorative overlays — native scroll semantics are maintained underneath
- Touch-friendly with `touch-none` on scrollbar handles to avoid conflicts
