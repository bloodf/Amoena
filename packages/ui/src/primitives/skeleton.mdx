# Skeleton

A placeholder loading indicator that mimics the shape of content before it loads. Uses a pulsing animation over a muted background.

## Import

```tsx
import { Skeleton } from "@lunaria/ui/atoms";
```

## Accessibility

- Rendered as a `<div>` with `animate-pulse` — purely visual with no semantic meaning
- Screen readers ignore skeleton elements by default; ensure the loading state is communicated via an `aria-busy` attribute or a visually hidden "Loading…" label on the parent container
- Replace skeletons with actual content as soon as data is available
