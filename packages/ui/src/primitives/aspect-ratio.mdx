# AspectRatio

A container that enforces a fixed width-to-height ratio for its content. Built on Radix UI's AspectRatio primitive — useful for images, videos, and embedded media.

## Import

```tsx
import { AspectRatio } from "@lunaria/ui/atoms";
```

## Accessibility

- The component is a layout wrapper with no semantic role — it does not affect screen reader output
- Ensure child content (images, videos) has appropriate `alt` text or `aria-label` attributes
- The aspect ratio is maintained via CSS `padding-bottom` trick, so content reflows correctly at all viewport sizes
