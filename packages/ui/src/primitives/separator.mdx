# Separator

A visual divider between content sections. Built on Radix UI's Separator primitive, supporting both horizontal and vertical orientations.

## Import

```tsx
import { Separator } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-separator` which renders with `role="separator"` (or `role="none"` when `decorative` is `true`)
- `decorative` defaults to `true`, hiding the separator from screen readers when it is purely visual
- Set `decorative={false}` when the separator conveys meaningful structure
