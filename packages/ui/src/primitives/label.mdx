# Label

An accessible label for form controls. Built on Radix UI's Label primitive, it automatically adjusts styling when its associated input is disabled.

## Import

```tsx
import { Label } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-label` which renders a native `<label>` element
- Use `htmlFor` to associate the label with a form control for screen readers and click-to-focus behavior
- Automatically shows disabled styling (`opacity-70`, `cursor-not-allowed`) when the peer input is disabled
