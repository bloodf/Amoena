# SearchField

A search input with a leading search icon and a clear button that appears when the field has a value. Wraps the base `Input` component.

## Import

## Accessibility

- The clear button includes `aria-label="Clear search"` for screen readers
- Search icon is decorative (`pointer-events-none`) and does not receive focus
- Standard `<input>` keyboard behavior — type to search, clear button is focusable via `Tab`
- Provide an `onClear` callback for custom clear behavior, or the default resets the value to empty
