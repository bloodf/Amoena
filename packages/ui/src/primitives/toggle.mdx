# Toggle

A two-state button that can be toggled on or off. Built on Radix UI's Toggle primitive with support for variant and size props.

## Import

```tsx
import { Toggle } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-toggle` which provides `aria-pressed` state management
- Toggled via `Space` or `Enter` keys
- Focus ring visible on keyboard navigation
- Disabled state prevents interaction and reduces opacity
