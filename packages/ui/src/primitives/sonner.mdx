# Sonner

Toast notification system powered by the [Sonner](https://sonner.emilkowal.ski/) library. Provides success, error, info, warning, and promise-based toasts with theme-aware styling.

## Import

```tsx
import { Toaster, toast } from "@lunaria/ui/atoms";
```

## Usage

Place `<Toaster />` once in your app layout, then call `toast()` anywhere to show notifications.

## Accessibility

- Toasts are announced to screen readers via a live region
- Action buttons within toasts are keyboard-focusable
- Toasts auto-dismiss after a timeout; users can dismiss early by pressing `Escape` or clicking the close button
- `richColors` mode provides high-contrast semantic colors for each toast type
