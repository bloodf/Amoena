# Pagination

A navigation component for paging through content. Includes page links, previous/next buttons, and an ellipsis indicator for truncated page ranges.

## Import

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationPrevious,
  PaginationNext,
  PaginationEllipsis,
} from "@lunaria/ui/atoms";
```

## Accessibility

- Wraps a `<nav>` with `role="navigation"` and `aria-label="pagination"`
- Active page link uses `aria-current="page"`
- Previous/Next buttons include descriptive `aria-label` text
- Ellipsis is hidden from screen readers with `aria-hidden` and includes a `sr-only` "More pages" label
- Uses button styling variants (`ghost` for inactive, `outline` for active) for clear visual state
