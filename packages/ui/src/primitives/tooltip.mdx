# Tooltip

A popup that displays informational text when hovering or focusing on an element. Built on Radix UI Tooltip primitive.

## Import


## Accessibility

- Wraps Radix UI Tooltip which implements WAI-ARIA tooltip pattern
- Appears on hover and focus, dismissed with `Escape`
- Content uses `role="tooltip"` and is linked via `aria-describedby`
- `TooltipProvider` controls the global `delayDuration` and `skipDelayDuration`
- Supports `sideOffset` for consistent spacing from the trigger
