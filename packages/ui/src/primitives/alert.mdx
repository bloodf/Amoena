# Alert

A static, prominent message block for displaying important information or feedback. Supports `default` and `destructive` variants.

## Import

```tsx
import { Alert, AlertTitle, AlertDescription } from "@lunaria/ui/atoms";
```

## Accessibility

- Uses `role="alert"` to announce content to screen readers immediately
- Icons are positioned absolutely and sibling text is offset with padding
- Use `AlertTitle` for a concise heading and `AlertDescription` for details
- The `destructive` variant applies semantic color cues for error states
