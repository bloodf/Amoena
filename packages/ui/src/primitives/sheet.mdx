# Sheet

A panel that slides in from any edge of the screen, built on Radix UI Dialog primitives. Supports four directions — `top`, `right`, `bottom`, and `left` — and is commonly used for navigation drawers, settings panels, or detail views.

## Import

```tsx
import {
  Sheet,
  SheetTrigger,
  SheetContent,
  SheetHeader,
  SheetFooter,
  SheetTitle,
  SheetDescription,
  SheetClose,
  SheetOverlay,
  SheetPortal,
} from "@lunaria/ui/atoms";
```

## Anatomy

```tsx
<Sheet>
  <SheetTrigger />
  <SheetContent side="right">
    <SheetHeader>
      <SheetTitle />
      <SheetDescription />
    </SheetHeader>
    {/* Body content */}
    <SheetFooter />
  </SheetContent>
</Sheet>
```

### SheetContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"top" \| "right" \| "bottom" \| "left"` | `"right"` | Which edge of the viewport the sheet slides in from |

## Accessibility

- Focus is trapped inside the sheet when open
- Pressing `Escape` closes the sheet
- A close button with a screen-reader-only label is rendered in the top-right corner
- `SheetTitle` maps to `aria-labelledby` and `SheetDescription` to `aria-describedby`
- Focus returns to the trigger when the sheet is closed
- The overlay blocks interaction with background content
