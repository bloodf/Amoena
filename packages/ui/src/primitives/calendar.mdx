# Calendar

A date picker calendar built on [react-day-picker](https://react-day-picker.js.org/). Supports single date selection and date range selection with styled navigation and outside-day display.

## Import

```tsx
import { Calendar } from "@lunaria/ui/atoms";
```

## Accessibility

- Fully keyboard navigable — arrow keys move between days, months navigated via previous/next buttons
- Selected days use `aria-selected` for screen reader announcements
- Outside days are visually muted and have reduced opacity
- Disabled days are marked with `aria-disabled`
- Navigation buttons include accessible icon components with clear visual indicators
