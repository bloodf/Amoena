# Checkbox

A control that allows the user to toggle between checked and unchecked states. Built on Radix UI's Checkbox primitive with a Lucide check icon indicator.

## Import

```tsx
import { Checkbox } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-checkbox` which provides `role="checkbox"` and manages `aria-checked` state
- Supports keyboard toggle via `Space` key
- Focus ring visible on keyboard navigation
- Disabled state prevents interaction and reduces opacity
- Pair with `<Label>` for a visible accessible label
