## Variants

| --------- | ----------------------------------------------- |
| `variant` | `default`, `secondary`, `destructive`, `outline` |

## Accessibility

- Renders as a `<div>` — add `role="status"` or wrap in an appropriate landmark if the badge conveys live status information
- Focus ring styles are included for use when the badge is interactive (e.g. as a link child)
- Use sufficient color contrast between variant background and foreground for WCAG compliance
