# Slider

A draggable range input for selecting a numeric value within a range. Built on Radix UI's Slider primitive with track, range, and thumb sub-components.

## Import

```tsx
import { Slider } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-slider` which provides `role="slider"` with `aria-valuemin`, `aria-valuemax`, and `aria-valuenow`
- Arrow keys adjust the value by `step`; `Home`/`End` jump to min/max
- Touch-friendly with `touch-none` to prevent scroll interference
- Focus ring visible on keyboard navigation
- Disabled state prevents interaction and reduces opacity
