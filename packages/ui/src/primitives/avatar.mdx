# Avatar

A circular image element for representing users or entities. Built on Radix UI's Avatar primitive with image and fallback support.

## Import

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-avatar` which renders an `<img>` with proper `alt` text support
- `AvatarFallback` automatically displays when the image fails to load, ensuring content is never blank
- The fallback shows initials or an icon, maintaining visual context for screen reader users
- Use a descriptive `alt` on `AvatarImage` for screen reader accessibility
