# Card

A container component for grouping related content with a bordered, rounded surface. Composed of `CardHeader`, `CardTitle`, `CardDescription`, `CardContent`, and `CardFooter` sub-components.

## Import

```tsx
import { Card, CardHeader, CardTitle, CardDescription, CardContent, CardFooter } from "@lunaria/ui/atoms";
```

## Accessibility

- Renders semantic HTML (`<div>`, `<h3>`, `<p>`) — use heading sub-components for proper document outline
- No interactive behavior by default; wrap in links or buttons for clickable cards
- Uses `bg-card` and `text-card-foreground` tokens for consistent theming
- All sub-components accept `className` for custom styling
