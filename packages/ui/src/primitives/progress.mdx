# Progress

A visual indicator showing the completion progress of a task. Built on Radix UI's Progress primitive with an animated fill bar.

## Import

```tsx
import { Progress } from "@lunaria/ui/atoms";
```

## Accessibility

- Built on `@radix-ui/react-progress` which provides `role="progressbar"` with `aria-valuenow`, `aria-valuemin`, and `aria-valuemax`
- Screen readers announce the current progress percentage
- The indicator uses a CSS `translateX` transform for smooth animated transitions
