# Input

A styled text input field. Extends the native `<input>` element with consistent styling, focus rings, and file-input support.

## Import

```tsx
import { Input } from "@lunaria/ui/atoms";
```

## Accessibility

- Renders a native `<input>` element preserving full keyboard and screen reader support
- Focus ring visible on keyboard navigation via `focus-visible` styles
- Disabled state shows `not-allowed` cursor and reduced opacity
- Pair with `<Label htmlFor>` to associate a visible label for screen readers
