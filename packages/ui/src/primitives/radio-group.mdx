A set of mutually exclusive options where only one can be selected at a time. Built on Radix UI's RadioGroup primitive.

## Accessibility

- Built on `@radix-ui/react-radio-group` which provides `role="radiogroup"` and `role="radio"` semantics
- Arrow keys navigate between options; `Space` selects the focused option
- Focus ring visible on keyboard navigation
- Disabled state prevents interaction and reduces opacity
- Pair each `RadioGroupItem` with a `<Label>` for screen reader support
