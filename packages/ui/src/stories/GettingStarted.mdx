# @lunaria/ui

**Lunaria** is a design system for AI-native developer tools. It provides a complete component library built on **Radix UI** primitives, **Tailwind CSS**, and **React 19**, with a dark-first HSL color system and design tokens managed via `@lunaria/tokens`.

---

## Installation

The package is a workspace dependency within the Lunaria monorepo:

```json
{
  "dependencies": {
    "@lunaria/ui": "workspace:*",
    "@lunaria/tokens": "workspace:*"
  }
}
```

## Importing

```tsx
// Import globals.css for base styles and CSS custom properties
import "@lunaria/ui/styles";

// Import primitives (shadcn primitives)
import { Button, Input, Card } from "@lunaria/ui/primitives";

// Import individual primitives
import { Badge } from "@lunaria/ui/primitives/badge";

// Import composite components
import { SessionComposer } from "@lunaria/ui/composites/SessionComposer";

// Import layout patterns
import { Screen } from "@lunaria/ui/patterns";

// Import hooks
import { useToast } from "@lunaria/ui/hooks/use-toast";

// Import utilities
import { cn } from "@lunaria/ui/lib/utils";
```

### Atoms

All **shadcn/ui** primitives live here — buttons, inputs, dialogs, tooltips, tabs, forms, select, accordion, and more. These are the building blocks used by every composite component.

### Composites

Feature-level components that combine primitives for specific application screens: session management, agent teams, command palette, marketplace, memory browser, terminal panels, and more.

### Patterns

Layout utilities that wrap content in standardized structures — full-screen shells, control bars, and settings layouts.

## Theming

Lunaria defaults to dark mode. Apply the `.light` class to switch themes:

```tsx
<div className="light">
  {/* Light mode content */}
</div>
```

All colors are HSL-based CSS custom properties. See **Foundation/Colors** for the full token reference.
