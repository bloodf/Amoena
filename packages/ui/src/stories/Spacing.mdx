# Spacing

Lunaria uses a **4px base unit** spacing scale, consistent with Tailwind CSS defaults. The full scale is defined in `@lunaria/tokens` as `lunariaSpacingTokens`.

## Spacing Scale

| Token | Value | Pixels | Tailwind Class | Common Use |
|---|---|---|---|---|
| `0` | 0px | 0 | `p-0`, `m-0`, `gap-0` | Reset spacing |
| `px` | 1px | 1 | `p-px`, `m-px` | Hairline borders/offsets |
| `0.5` | 2px | 2 | `p-0.5`, `gap-0.5` | Tight inline spacing |
| `1` | 4px | 4 | `p-1`, `gap-1` | Minimal padding, icon gaps |
| `1.5` | 6px | 6 | `p-1.5`, `gap-1.5` | Small element padding |
| `2` | 8px | 8 | `p-2`, `gap-2` | Standard small gap |
| `2.5` | 10px | 10 | `p-2.5`, `gap-2.5` | Between small + standard |
| `3` | 12px | 12 | `p-3`, `gap-3` | Standard element padding |
| `3.5` | 14px | 14 | `p-3.5` | Between standard gaps |
| `4` | 16px | 16 | `p-4`, `gap-4` | Standard section padding |
| `5` | 20px | 20 | `p-5`, `gap-5` | Comfortable padding |
| `6` | 24px | 24 | `p-6`, `gap-6` | Card padding, section gaps |
| `7` | 28px | 28 | `p-7` | Between standard + large |
| `8` | 32px | 32 | `p-8`, `gap-8` | Large section padding |
| `9` | 36px | 36 | `p-9` | Between large sizes |
| `10` | 40px | 40 | `p-10`, `gap-10` | Major section spacing |
| `11` | 44px | 44 | `p-11` | Touch target height (44px) |
| `12` | 48px | 48 | `p-12`, `gap-12` | Large layout spacing |
| `14` | 56px | 56 | `p-14` | Extra-large spacing |
| `16` | 64px | 64 | `p-16`, `gap-16` | Page-level spacing |
| `20` | 80px | 80 | `p-20` | Hero/section spacing |
| `24` | 96px | 96 | `p-24` | Maximum layout spacing |

## Common Patterns

### Component Internal Spacing

```tsx
{/* Button padding */}
<button className="px-4 py-2">Action</button>

{/* Card layout */}
<div className="p-6 space-y-4">
  <h3>Title</h3>
  <p>Content</p>
</div>

{/* Form field gap */}
<div className="flex flex-col gap-2">
  <label>Email</label>
  <input />
</div>
```

### Layout Spacing

```tsx
{/* Page container — defined in tailwind.config.ts */}
{/* container: center: true, padding: "2rem", max-width 2xl: 1400px */}

{/* Section spacing */}
<section className="py-8 space-y-6">...</section>

{/* Sidebar + content */}
<div className="flex gap-6">
  <aside className="w-64 p-4">...</aside>
  <main className="flex-1 p-6">...</main>
</div>
```

### Recommended Spacing by Context

| Context | Spacing | Example |
|---|---|---|
| Icon + label gap | `gap-1` to `gap-2` (4–8px) | `flex items-center gap-2` |
| Form field stacking | `gap-2` (8px) | `flex flex-col gap-2` |
| Button group | `gap-2` (8px) | `flex gap-2` |
| Card content padding | `p-4` to `p-6` (16–24px) | `p-6` |
| Section vertical rhythm | `space-y-4` to `space-y-6` | `space-y-6` |
| Page sections | `py-8` to `py-12` (32–48px) | `py-8` |

## Border Radius

Defined via the `--radius` CSS variable (default `0.25rem`) with calculated variants in `tailwind.config.ts`:

| Token | Value | Tailwind Class |
|---|---|---|
| `sm` | `calc(var(--radius) - 4px)` | `rounded-sm` |
| `md` | `calc(var(--radius) - 2px)` | `rounded-md` |
| `lg` | `var(--radius)` (0.25rem) | `rounded-lg` |

The full radius scale from `@lunaria/tokens`:

| Token | Value |
|---|---|
| `none` | 0 |
| `sm` | 0.125rem |
| `DEFAULT` | 0.25rem |
| `md` | 0.375rem |
| `lg` | 0.5rem |
| `xl` | 0.75rem |
| `2xl` | 1rem |
| `3xl` | 1.5rem |
| `full` | 9999px |
